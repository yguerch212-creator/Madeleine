⚙️ Configuration WordPress — À faire par le groupe

DreamBox — Groupe 2 (Madeleine) — Les étapes à faire directement sur le site

Ce document liste tout ce qui doit être fait directement dans WordPress. Les pages (Expérience + Politique de Confidentialité) et le cahier des charges sont déjà prêts dans le repo. Ce PDF couvre le reste.

1 Installer et configurer Yoast SEO

Si Yoast SEO n'est pas encore installé :

  1. Aller dans Extensions → Ajouter
  2. Chercher « Yoast SEO »
  3. Cliquer Installer puis Activer

Configurer le SEO de chaque page :

Pour CHAQUE page du site, ouvrir la page dans l'éditeur WordPress (pas Elementor), descendre en bas jusqu'au bloc Yoast SEO, et remplir :

Page Accueil :

  • Titre SEO : DreamBox — Et si tu pouvais choisir tes rêves ?
  • Meta description : DreamBox vous permet de voyager dans vos univers préférés pendant votre sommeil. Films, séries, animés — chaque nuit devient une aventure.

Page Stations de Rêve :

  • Titre SEO : Stations de Rêve — Catalogue des univers DreamBox
  • Meta description : Explorez notre catalogue de cassettes de rêve : films, séries, animés et dessins animés. Choisissez votre prochaine aventure nocturne.

Page Nos Machines :

  • Titre SEO : Nos Machines DreamBox — La technologie du rêve
  • Meta description : Découvrez les machines DreamBox : fonctionnement, packs disponibles et prix. Transformez votre sommeil en expérience immersive.

Page L'Expérience :

  • Titre SEO : L'Expérience DreamBox — FAQ, Témoignages & Qualité du Sommeil
  • Meta description : Découvrez les témoignages de nos rêveurs, notre FAQ complète et nos garanties sur la qualité de votre sommeil avec DreamBox.

Page Politique de Confidentialité :

  • Titre SEO : Politique de Confidentialité — DreamBox
  • Meta description : Consultez la politique de confidentialité de DreamBox. Protection de vos données personnelles conformément au RGPD.

2 Installer le bandeau de cookies

  1. Aller dans Extensions → Ajouter
  2. Chercher « CookieYes » (gratuit et simple)
  3. Cliquer Installer puis Activer
  4. Un assistant de configuration va apparaître, suivez les étapes
  5. Le bandeau s'affichera automatiquement sur toutes les pages

Alternative : « Complianz » est aussi très bien si CookieYes pose problème.

3 Ajouter le favicon

Le favicon c'est la petite icône qui s'affiche dans l'onglet du navigateur.

  1. Aller dans Apparence → Personnaliser
  2. Cliquer sur Identité du site
  3. Tout en bas, cliquer sur « Sélectionner une icône du site »
  4. Importer une image carrée (512×512 pixels idéalement)
    Idée : un croissant de lune, une boîte avec des étoiles, ou le logo DreamBox
  5. Cliquer Publier

4 Créer le compte admin du prof

Le prof doit pouvoir se connecter. Vérifiez d'abord si le compte existe :

  1. Aller dans Utilisateurs → Tous les utilisateurs
  2. Chercher « steven »

S'il n'existe pas :

  1. Aller dans Utilisateurs → Ajouter
  2. Identifiant : steven
  3. Adresse email : formation@stevenberg.fr
  4. Mot de passe : 00099 (décocher "envoyer un email" et cocher "définir le mot de passe")
  5. Rôle : Administrateur
  6. Cliquer Ajouter un utilisateur

5 Lier la Politique de Confidentialité dans le footer

Quand on clique sur "Politique de Confidentialité" dans le footer, ça doit mener vers la page.

Si le footer est fait avec Elementor :

  1. Aller dans Templates → Theme Builder (ou trouver le template de footer)
  2. Modifier le footer avec Elementor
  3. Ajouter un widget Texte ou Menu de navigation
  4. Ajouter un lien vers la page Politique de Confidentialité
  5. Sauvegarder

Si le footer est dans le menu WordPress :

  1. Aller dans Apparence → Menus
  2. Sélectionner le menu du footer (ou en créer un)
  3. Ajouter la page « Politique de Confidentialité »
  4. Sauvegarder le menu

6 Remplir les attributs alt des images

Chaque image du site doit avoir un texte alternatif (pour le SEO et l'accessibilité).

  1. Aller dans Médias → Bibliothèque
  2. Cliquer sur chaque image, une par une
  3. Dans le panneau à droite, remplir le champ « Texte alternatif »

Exemples :

  • Image de la DreamBox → Machine DreamBox posée sur une table de nuit
  • Image d'un univers film → Univers de rêve inspiré du film Inception
  • Logo → Logo DreamBox
  • Image ambiance nuit → Ambiance nocturne étoilée DreamBox

7 Vérifier le responsive

Chaque page doit s'afficher correctement sur ordinateur, tablette et mobile.

  1. Ouvrir chaque page avec Elementor
  2. En bas à gauche de l'éditeur, cliquer sur l'icône 📱
  3. Basculer entre Desktop, Tablette et Mobile
  4. Vérifier que rien ne déborde ou ne se chevauche
  5. Si un truc bug : ajuster la taille du texte ou les marges pour ce format

8 Exporter le site (rendu final)

Le prof demande un fichier .wpress :

  1. Aller dans Extensions → Ajouter
  2. Chercher « All-in-One WP Migration »
  3. Installer et activer
  4. Dans le menu à gauche : All-in-One WP Migration → Exporter
  5. Cliquer sur « Exporter vers → Fichier »
  6. Attendre que le fichier se génère (ça peut prendre 1-2 minutes)
  7. Télécharger le fichier .wpress

Capture d'écran desktop :

  1. Ouvrir chaque page dans le navigateur (en taille desktop, plein écran)
  2. Faire une capture d'écran (touche Impr. Écran ou outil Capture Windows)
  3. Sauvegarder comme image (.png ou .jpg)

Dossier de rendu final :

Créer un dossier avec vos noms et mettre dedans :

  • Le fichier .wpress
  • Les captures d'écran desktop de chaque page
  • Le cahier-des-charges.pdf

Déposer le tout dans la dropbox indiquée par le prof.

✅ Checklist finale

  • Yoast SEO installé et titres/descriptions remplis pour chaque page
  • Bandeau cookies installé (CookieYes ou Complianz)
  • Favicon ajouté
  • Compte admin "steven" créé
  • Lien "Politique de Confidentialité" dans le footer
  • Attributs alt remplis sur toutes les images
  • Pages responsive (vérifiées desktop / tablette / mobile)
  • Export .wpress téléchargé
  • Captures d'écran desktop faites
  • Cahier des charges dans le dossier de rendu
  • Dossier déposé dans la dropbox
