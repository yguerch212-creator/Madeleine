## 📦 Contenu du repo

```
├── elementor-templates/
│   ├── page-experience.json              ← Page "L'Expérience" (à importer dans Elementor)
│   └── page-politique-confidentialite.json ← Page "Politique de Confidentialité" (à importer)
├── cahier-des-charges.pdf                ← Cahier des charges du projet (livrable noté)
├── guide-complet-dreambox.pdf            ← Guide complet pas-à-pas (import + config + checklist)
└── README.md                             ← Ce fichier
```

## 🚀 Comment utiliser

1. Lire le **guide-complet-dreambox.pdf** — tout est expliqué dedans, étape par étape
2. Importer les fichiers `.json` dans Elementor (Templates → Mes templates → Importer)
3. Suivre la checklist du guide pour ne rien oublier avant le rendu
