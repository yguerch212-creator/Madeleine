## 🚀 Import rapide

1. Télécharge les fichiers `.json` du dossier `elementor-templates/`
2. Dans WordPress : **Elementor → Mes templates → Importer des templates**
3. Sélectionne le fichier JSON et importe
4. Crée une nouvelle page → **Modifier avec Elementor** → 📁 → **Mes templates** → Insérer

👉 **Guide complet** : voir `guide-import-elementor.pdf`
