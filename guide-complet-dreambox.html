Partie 1 — C'est quoi les fichiers ?

En résumé : Le dossier contient des pages toutes faites que vous pouvez importer dans WordPress en 2 clics. Pas besoin de coder, pas besoin de CSS, tout se fait dans Elementor.

📁 Les fichiers du dossier

elementor-templates/page-experience.json

C'est la page "L'Expérience" toute faite. Elle contient :

  • Un grand titre en haut
  • 6 avis de clients (avec des étoiles et des citations)
  • 3 blocs de réassurance sur la qualité du sommeil
  • Une FAQ avec 8 questions/réponses (qui s'ouvrent en cliquant dessus)
  • Un bouton "Découvrir nos machines" en bas

👉 C'est un fichier .json = un format que Elementor sait lire et transformer en page automatiquement.

elementor-templates/page-politique-confidentialite.json

C'est la page "Politique de Confidentialité" toute faite. Le truc légal qu'on doit mettre dans le footer.

Elle contient tout ce qu'il faut : collecte de données, RGPD, cookies, droits des utilisateurs, etc.

👉 Même principe : fichier .json importable dans Elementor.

cahier-des-charges.pdf

C'est le cahier des charges du projet (demandé dans les consignes, ça vaut 1 point). Il explique :

  • C'est quoi DreamBox, la mission, les valeurs
  • Le public visé
  • Les objectifs du site
  • La structure des pages
  • La charte graphique (couleurs, typo, style)
  • Les contraintes techniques

👉 C'est un PDF, vous avez juste à le mettre dans le dossier de rendu.

Ce guide (guide-complet-dreambox.pdf)

Partie 2 — Comment importer les pages

⚠️ Pas de panique : L'import ne touche PAS aux pages qui existent déjà. On va d'abord créer des pages de test pour voir le résultat, et remplacer seulement quand on est contente.

1 Se connecter à WordPress

  1. Ouvrir un navigateur et aller sur : groupe2.stevenberg.fr/wp-admin
  2. Se connecter avec vos identifiants

2 Importer les templates dans la bibliothèque

  1. Dans le menu à gauche, cliquer sur « Templates » puis « Mes templates »
    (Si vous voyez "Elementor" dans le menu, c'est là-dedans aussi)
  2. En haut de la page, cliquer sur le bouton « Importer des templates »
    (C'est une petite icône de flèche vers le haut ↑)
  3. Cliquer sur « Choisir un fichier »
  4. Aller chercher le fichier page-experience.json
  5. Cliquer sur « Importer maintenant »
  6. Recommencer la même chose pour page-politique-confidentialite.json
✅ C'est fait ! Les deux templates sont maintenant dans votre bibliothèque. Rien n'a changé sur le site, c'est juste stocké dans Elementor.

3 Créer une page de test pour voir le résultat

  1. Aller dans Pages → Ajouter
  2. Mettre comme titre : [TEST] L'Expérience
  3. Cliquer sur « Modifier avec Elementor » (le gros bouton bleu)
  4. Une fois dans Elementor, cliquer sur l'icône dossier 📁 au milieu de la page blanche
  5. Aller dans l'onglet « Mes templates »
  6. Trouver « L'Expérience » dans la liste
  7. Cliquer sur « Insérer »
  8. Si un popup demande "Voulez-vous importer les paramètres du document ?" → cliquer « Oui »
  9. La page apparaît ! 🎉
  10. Cliquer sur « Sauvegarder en brouillon » (en bas à gauche, la petite flèche à côté de Publier)

Faire pareil pour la politique de confidentialité : créer une page [TEST] Politique de Confidentialité, insérer le template.

4 Comparer et remplacer

  1. Dans la liste des pages (Pages → Toutes les pages), survoler la page [TEST] et cliquer « Aperçu »
  2. Ouvrir la page existante dans un autre onglet pour comparer
  3. Si c'est bon 👍 :

Pour remplacer :

  1. Mettre l'ancienne page à la corbeille
  2. Enlever "[TEST]" du titre de la nouvelle page
  3. Vérifier que l'URL (le "slug") est correct :
    → experience pour la page Expérience
    → politique-de-confidentialite pour la politique
  4. Cliquer « Publier »

Partie 3 — Les trucs à faire sur WordPress

Ces trucs-là sont demandés dans les consignes du prof et il faut les faire directement dans WordPress :

🔗 Lier la Politique de Confidentialité dans le footer

Quand quelqu'un clique sur "Politique de Confidentialité" dans le footer, ça doit mener vers la page.

  1. Ouvrir le footer avec Elementor (si c'est un template Elementor)
    Ou bien : Apparence → Menus
  2. Ajouter un lien vers la page Politique de Confidentialité
  3. Sauvegarder

🔍 Configurer le SEO (Yoast SEO)

Si le plugin Yoast SEO est installé, pour chaque page :

  1. Ouvrir la page dans l'éditeur WordPress (pas Elementor)
  2. Descendre en bas, y'a un bloc Yoast SEO
  3. Remplir le Titre SEO et la Meta description

Pour la page Expérience :

  • Titre : L'Expérience DreamBox — FAQ, Témoignages & Qualité du Sommeil
  • Description : Découvrez les témoignages de nos rêveurs, notre FAQ complète et nos garanties sur la qualité de votre sommeil avec DreamBox.

Pour la Politique de Confidentialité :

  • Titre : Politique de Confidentialité — DreamBox
  • Description : Consultez la politique de confidentialité de DreamBox. Protection de vos données personnelles conformément au RGPD.

⚠️ Si Yoast n'est pas installé : Extensions → Ajouter, chercher "Yoast SEO", installer, activer.

🍪 Installer le bandeau cookies

  1. Aller dans Extensions → Ajouter
  2. Chercher « CookieYes » ou « Complianz » (les deux sont gratuits)
  3. Installer et activer
  4. Suivre l'assistant de configuration (c'est guidé)
  5. Le bandeau apparaîtra automatiquement sur toutes les pages

⭐ Favicon (la petite icône dans l'onglet du navigateur)

  1. Aller dans Apparence → Personnaliser → Identité du site
  2. Tout en bas : « Icône du site »
  3. Mettre une image carrée (512x512 px idéalement) — un petit logo DreamBox ou un croissant de lune
  4. Publier

👤 Vérifier le compte admin du prof

Le prof doit pouvoir se connecter avec ces identifiants :

  • Email : formation@stevenberg.fr
  • Identifiant : steven
  • Mot de passe : 00099

Pour vérifier : Utilisateurs → Tous les utilisateurs. Si "steven" n'existe pas :

  1. Utilisateurs → Ajouter
  2. Remplir avec les infos ci-dessus
  3. Rôle : Administrateur
  4. Cliquer « Ajouter un utilisateur »

🖼️ Vérifier les attributs alt des images

Toutes les images du site doivent avoir un texte alternatif (c'est pour le SEO et l'accessibilité).

  1. Aller dans Médias → Bibliothèque
  2. Cliquer sur chaque image
  3. Remplir le champ « Texte alternatif » avec une courte description
    Exemple : "Boîte à rêves DreamBox sur table de nuit" ou "Logo DreamBox"

Partie 4 — Checklist avant de rendre

💡 Cochez chaque case une fois que c'est fait :

📄 Pages

  • Page Accueil OK et publiée
  • Page Stations de Rêve (Catalogue) OK et publiée
  • Page Nos Machines OK et publiée
  • Page L'Expérience importée et publiée
  • Page Politique de Confidentialité importée et publiée
  • Lien "Politique de Confidentialité" dans le footer

🔍 SEO

  • Yoast SEO installé et activé
  • Titre SEO rempli sur chaque page
  • Meta description remplie sur chaque page
  • 1 seul H1 par page (déjà respecté dans les templates)
  • Attributs alt renseignés sur toutes les images

🎨 Design

  • Favicon présent (petite icône dans l'onglet)
  • Les pages s'affichent bien sur ordinateur
  • Les pages s'affichent bien sur tablette
  • Les pages s'affichent bien sur mobile

🔧 Technique

  • Bandeau cookies installé et fonctionnel
  • Compte admin "steven" créé (formation@stevenberg.fr / steven / 00099)
  • Thème Hello Elementor actif

📦 Rendu final

  • Export .wpress créé (plugin All-in-One WP Migration → Exporter → Fichier)
  • Capture d'écran desktop de chaque page
  • Cahier des charges (cahier-des-charges.pdf) dans le dossier
  • Noms des étudiants sur le dossier
  • Déposer dans la dropbox

🚀 Pour l'export .wpress

  1. Aller dans Extensions → Ajouter
  2. Chercher « All-in-One WP Migration »
  3. Installer et activer
  4. Dans le menu à gauche : All-in-One WP Migration → Exporter
  5. Cliquer sur « Exporter vers → Fichier »
  6. Attendre que le fichier se génère
  7. Télécharger le fichier .wpress
