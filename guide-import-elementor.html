A. Contenu du dossier

Le dossier elementor-templates/ contient :

  • page-experience.json — La page "L'Expérience" (FAQ + retours clients + réassurance sommeil)
  • page-politique-confidentialite.json — La page "Politique de Confidentialité"

Chaque fichier est un template Elementor importable directement dans WordPress.

B. Méthode 1 — Import comme Template (Recommandé)

Cette méthode ajoute les pages dans ta bibliothèque de templates sans toucher au site existant.

Étape 1 : Accéder à la bibliothèque de templates

  1. Connecte-toi au tableau de bord WordPress : groupe2.stevenberg.fr/wp-admin
  2. Dans le menu de gauche, va dans Templates → Templates enregistrés (ou Elementor → Mes templates)

Étape 2 : Importer le template

  1. En haut de la page, clique sur « Importer des templates » (icône flèche vers le haut)
  2. Clique sur « Choisir un fichier »
  3. Sélectionne le fichier JSON (ex: page-experience.json)
  4. Clique sur « Importer maintenant »
✅ Résultat : Le template apparaît dans ta bibliothèque. Aucune page existante n'est modifiée.

Étape 3 : Prévisualiser le template

  1. Dans la bibliothèque de templates, survole le template importé
  2. Clique sur « Aperçu » pour voir le rendu
  3. Tu peux comparer visuellement avec ta page existante dans un autre onglet

C. Méthode 2 — Créer une nouvelle page puis insérer le template

Si tu préfères créer une page de test d'abord :

Étape 1 : Créer une page brouillon

  1. Va dans Pages → Ajouter
  2. Donne un titre temporaire, ex: « [TEST] L'Expérience »
  3. Clique sur « Modifier avec Elementor »

Étape 2 : Insérer le template importé

  1. Dans l'éditeur Elementor, clique sur l'icône dossier (📁) au centre de la page
  2. Va dans l'onglet « Mes templates »
  3. Trouve le template importé (ex: « L'Expérience »)
  4. Clique sur « Insérer »
  5. Si une popup demande « Voulez-vous importer les paramètres du document ? », clique « Oui »

Étape 3 : Vérifier et publier en brouillon

  1. Vérifie que le design s'affiche correctement
  2. Teste le responsive : dans Elementor, en bas à gauche, clique sur l'icône 📱 pour basculer entre desktop/tablette/mobile
  3. Sauvegarde en brouillon (ne publie pas encore)
  4. Tu peux comparer avec ta page existante dans un autre onglet

D. Visualiser avant de remplacer

💡 Astuce : Avec la méthode 2, ta page de test est en brouillon. Tu peux la prévisualiser sans qu'elle soit visible publiquement. Compare-la avec la page existante avant de décider.

Pour prévisualiser :

  1. Dans la liste des pages, survole ta page « [TEST] L'Expérience »
  2. Clique sur « Aperçu »
  3. Ouvre la page existante dans un autre onglet pour comparer

E. Remplacer la page existante (quand tu es prête)

⚠️ Fais ceci uniquement quand tu es satisfaite du résultat !

Option A : Remplacer le contenu de la page existante

  1. Ouvre la page existante avec Elementor
  2. Sélectionne tout le contenu (clic droit sur une section → « Sélectionner tout »)
  3. Supprime tout (touche Suppr)
  4. Clique sur l'icône dossier 📁 → Mes templates → Insère le template
  5. Publie

Option B : Supprimer l'ancienne et renommer la nouvelle

  1. Mets l'ancienne page dans la corbeille
  2. Renomme ta page [TEST] en enlevant « [TEST] »
  3. Vérifie que le slug (URL) est correct (ex: /experience/)
  4. Publie

F. Spécificités par page

Page « L'Expérience »

  • Slug recommandé : /experience/
  • Contenu : Section hero, 6 témoignages clients, 3 piliers de réassurance sommeil, FAQ 8 questions, CTA final
  • SEO (Yoast) : Titre : « L'Expérience DreamBox — FAQ, Témoignages & Qualité du Sommeil » / Meta description : « Découvrez les témoignages de nos rêveurs, notre FAQ complète et nos garanties sur la qualité de votre sommeil avec DreamBox. »

Page « Politique de Confidentialité »

  • Slug recommandé : /politique-de-confidentialite/
  • Contenu : 12 sections RGPD complètes
  • Lien footer : Ajouter un lien dans le footer vers cette page
  • SEO (Yoast) : Titre : « Politique de Confidentialité — DreamBox » / Meta description : « Consultez la politique de confidentialité de DreamBox. Informations sur la collecte, l'utilisation et la protection de vos données personnelles. »

G. Lier la Politique de Confidentialité dans le Footer

  1. Va dans Apparence → Menus (ou édite le footer avec Elementor)
  2. Ajoute un lien vers la page « Politique de Confidentialité »
  3. Si le footer est fait avec Elementor : ouvre le template de footer, ajoute un widget Texte ou Menu avec le lien vers /politique-de-confidentialite/

H. Dépannage

ProblèmeSolution
Le template ne s'importe pasVérifie que le fichier est bien en .json et non .json.txt. Vérifie qu'Elementor est activé.
Les couleurs sont différentesVa dans Elementor → Réglages du site → Couleurs globales et vérifie que le fond du site est #2C2F2F.
La police Poppins ne s'affiche pasPoppins est une Google Font, elle devrait se charger automatiquement. Sinon : Elementor → Réglages → Typographie → Poppins.
Le toggle FAQ ne s'ouvre pasAssure-toi d'utiliser Elementor (pas Gutenberg). Le widget Toggle est natif dans Elementor.
Les étoiles des avis ne s'affichent pasLe widget Star Rating nécessite Elementor Pro. Alternative : remplacer par des ⭐ en texte.
